# Blog Website
A website where blog readers and writers would love to be!!

**Technologies used**
  * Front End : HTML, CSS, BootStrap
  * Back End : Node JS
  * Framework : Express JS
  * Database : MongoDB
 
 ## Steps to build:
 1. `git clone https://github.com/Swastyy/Interview-prep-tracker`
 2. `cd Interview-prep-tracker`
 3. `npm install`
 4. `npm start`
